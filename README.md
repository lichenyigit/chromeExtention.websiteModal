# 流媒体网站字幕遮幕

Chrome 扩展，用于在视频网站上添加遮幕，遮挡字幕区域。

## 功能

- 点击扩展图标切换遮幕显示/隐藏
- 遮幕可拖拽移动
- 遮幕可调整大小（拖拽右下角）
- 自动记住每个网站的遮幕位置
- 自动检测视频元素并定位到字幕区域

## 安装

1. 下载本项目
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## 使用方法

1. 打开任意视频网站
2. 点击浏览器工具栏的扩展图标，显示遮幕
3. 拖拽遮幕到字幕位置
4. 拖拽遮幕右下角调整大小
5. 再次点击扩展图标隐藏遮幕
